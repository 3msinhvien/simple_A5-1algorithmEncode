# A5/1 Audio Encryption/Decryption
B1: Install các thư viện cần thiết 
- Mở terminal, chạy lệnh sau: pip install -r requirement.txt
B2: Chạy chương trình bằng lệnh: python ui.py
B3: Nhập khóa vào ô giao diện (Key mẫu: mysecretkey123)
B4: Chọn file âm thanh (định dạng WAV) cần mã hóa hoặc giải mã. (File mẫu mình đã để cùng với mã nguồn)
B5: Chọn mã hoá/ giải mã, file output sẽ được lưu cùng mã nguồn của chương trình
